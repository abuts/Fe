EnCuts_fileDescription

The file is generated by @cuts_le_processor class method refit_sw_findJ
and contains the number of 1D constant energy cuts, extracted from sequence of TF_ files which are the
input data for @cuts_le_processor.

The file name consist of the following parts:
EnCuts_[Source SQW file name]_dE[cut's energy transfer]_dir_![the direction of the cut i.e. u-axis defining the cut in hkl]!.mat 
e.g. 
EnCuts_Fe_ei200_dE120_dir_!100!.mat
